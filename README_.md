# 🧹 CleanIQ

An AI-powered data cleaning and profiling platform built with Python, Streamlit, and Pandas.

## 🌐 Live Demo

https://cleaniq-ayush.streamlit.app/

## ✨ Features

- 📊 Dataset Profiling
- 🩺 Health Score
- 🧹 Automated Data Cleaning
- 🔍 Data Quality Assessment
- 📥 CSV Export
- 📊 Excel Export
- 🗄️ MySQL Export

## 🛠️ Tech Stack

- Python
- Streamlit
- Pandas
- NumPy
- SQLAlchemy
- OpenPyXL
- PyMySQL

## 🚀 How to Run Locally

```bash
git clone https://github.com/Ayushpatel3002/CleanIQ.git
cd CleanIQ
pip install -r requirements.txt
streamlit run app.py
```
